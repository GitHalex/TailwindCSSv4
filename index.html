Selectores basicos

Los selectores basicos son aquellos que permiten seleccionar elementos HTML de manera sencilla y directa. Algunos de los selectores basicos mas comunes son:

  • Selector de tipo: Selecciona todos los elementos de un tipo especifico, como todos los parrafos (p) o todos los encabezados (h1, h2, etc.). Ejemplo: p { color: blue; }
  • Selector de clase: Selecciona todos los elementos que tienen una clase especifica. Las clases se definen en el atributo class del elemento HTML. Ejemplo: .mi-clase { font-size: 16px; }
  • Selector de ID: Selecciona un elemento unico que tiene un ID especifico. Los IDs se definen en el atributo id del elemento HTML. Ejemplo: #mi-id { background-color: yellow; }
  • Selector universal: Selecciona todos los elementos en la pagina. Ejemplo: * { margin: 0; padding: 0; }
  • Selector de atributo: Selecciona elementos basados en la presencia o valor de un atributo especifico. Ejemplo: [type="text"] { border: 1px solid black; }

Estos son solo algunos ejemplos de selectores basicos en CSS. Combinando estos selectores, puedes crear reglas de estilo mas complejas y especificas para tus paginas web.
